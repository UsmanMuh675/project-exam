Description

This WeatherApp is a fully functional single-page application built with HTML, CSS, and JavaScript. It uses the Open-Meteo API (completely free, no API key required) to fetch and display real-time weather data for any city worldwide. The app first calls the Geocoding API to convert a city name into latitude and longitude coordinates, then uses those coordinates to retrieve current weather conditions and a 5-day forecast. The UI follows the project brief's design specifications with a navy header (#1A3C5E), teal hero section (#2E86AB), and responsive layout that works on both desktop and mobile devices. All JavaScript functions are clearly commented, and error handling is implemented for cases where a city is not found or API calls fail. The app includes a loading state, hover effects, and meets all technical requirements outlined in the project brief
